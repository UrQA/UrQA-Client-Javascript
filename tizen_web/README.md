Urqa-Cordova-Plugin
===================

Urqa-Cordova-Plugin V 0.1

# Menual Install

## copy to project ( work on UrQA-Client-Javascript folder )

install_tizen_web.sh [project root] [version]

    : version is release version ( check release folder and folder name is version ( ex> 0.0.2 ) )

# Sample index.html

```html
<!DOCTYPE html>
<html>

<head>
    <meta charset="utf-8"/>
    <meta name="description" content="A Tizen Web UI FW single-page template generated by Tizen Web IDE"/>
    <meta name="viewport" content="width=device-width,user-scalable=no"/>
    
    <link rel="stylesheet" href="tizen-web-ui-fw/latest/themes/tizen-white/tizen-web-ui-fw-theme.css" name="tizen-theme"/>
    <title>Tizen Web IDE - Template - Tizen - Tizen Web UI Framework - Single-Page</title>
    <!--NOTE:
        jquery.js and web-ui-fw.js must be included.
        DO NOT REMOVE below code!
    -->
    <script src="tizen-web-ui-fw/latest/js/jquery.js"></script>
    <script src="tizen-web-ui-fw/latest/js/tizen-web-ui-fw-libs.js"></script>
    <script src="tizen-web-ui-fw/latest/js/tizen-web-ui-fw.js"
        data-framework-theme="tizen-white"></script>

    <!--NOTE:
        Additional scripts and css files are to be placed here.
        You can use jQuery namespace($) and all functionalities in jQuery
        in your script.  For example:

            <script src="main.js"></script>
            <link rel="stylesheet" href="my.css">
    -->
    <script type="text/javascript" src="./js/main.js"></script>    
    <link rel="stylesheet" type="text/css" href="./css/style.css"/>

    <!-- urqa -->
    <script type="text/javascript" src="js/urqa-web-0.0.2.js"></script> 

</head>

<body>
    <div data-role="page">
        <div data-role="header" data-position="fixed">
            <h1>Single-page application </h1>
        </div><!-- /header -->

        <div data-role="content">
            <p>
                <a onclick="test_sendexception()">hello</a>         
            </p>
            <p>
                <a onclick="add_log()">add log</a>          
            </p>
        </div><!-- /content -->

        <div data-role="footer" data-position="fixed">
            <h4>Footer content</h4>
        </div><!-- /footer -->
    </div><!-- /page -->
    <script type="text/javascript" >

        // Init urqa
        urqa.Init( { api_key: '10Eb2BF7', app_version:"1.1", wrap_url:'http://192.168.9.6:3000/urqa_wrapper', console_debug_enable: true } );

        console.log( "start...");
        function test_sendexception(){
            console.log( "called test_sendexception --- 1");
            try{

                var test = null;

                test.aaa();

            }catch( err ){
                urqa.send_e( err, { errname: "hhh this is tizen test error", rank : 0,  tag: "tizen" } );
            }

            console.log( "test_sendexception end... -- 2");

            console.log( "global exception  --- 1");

            test.aaa();

            console.log( "global exception end... -- 2");
        }

        var log_test_cnt = 0;
        function add_log(){

            console.log( log_test_cnt );
            ++ log_test_cnt;

        }

    </script>
</body>
</html>
```
